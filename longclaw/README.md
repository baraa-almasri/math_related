# Longclaw

### Go to the [wiki page](https://github.com/baraa-almasri/longclaw/wiki) for class documentation 
### Prerequisites:

- git `duh`
- clang
- make
- gcc *****alternatively*****  


### Installing:

- clone the repo `git clone https://github.com/baraa-almasri/longclaw`
- compiling the source code:
  
     `make`
       
- or compile using gcc
     
     `make gcc`
